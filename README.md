# GreedyMonster-Easy2D

控制贪婪的小怪兽吃掉星星！

## 运行方法

1. 在安装了 Git 的终端上运行 `git clone https://github.com/Easy2D/GreedyMonster-Easy2D.git --recursive`
2. 打开项目目录下的 `.sln` 文件
3. 编译并启动！
